1. Difference between local storage and session storage ?

Ans:SessionStorage is similar to localStorage ; the difference is that while data in localStorage doesn't expire, data in sessionStorage is cleared when the page session ends. Whenever a document is loaded in a particular tab in the browser, a unique page session gets created and assigned to that particular tab

2. Difference between global scope and local scope in javascript ?

Ans:JavaScript variables have only two scopes. Global Variables A global variable has a global scope which means it can be defined anywhere in your JavaScript code. Local Variables A local variable will be visible only within a function where it is defined. Function parameters are always local to that function
